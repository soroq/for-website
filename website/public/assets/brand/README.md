# Shroq Brand Assets

This logo system is built around the current product truth: Shroq/Soroq is a Flutter OTA release-control platform focused on patches, rollback, and operator confidence.

## Concept

- The outer loop is the release cycle: shipped app, patch check, activation, rollback.
- The check mark is the verified patch path.
- The amber dot is the "dawn" cue behind the Shroq name and a small signal for a newly available update.
- The diagonal tail gives the mark a subtle `q` shape, so it can work for both the current repo spelling, `Soroq`, and the requested `Shroq` wordmark while naming settles.

## Files

- `shroq-mark.svg` - primary app/site mark with rounded-square background.
- `shroq-symbol.svg` - transparent symbol for places where the background is controlled elsewhere.
- `shroq-horizontal.svg` - primary Shroq wordmark for light backgrounds.
- `shroq-horizontal-dark.svg` - Shroq wordmark for dark backgrounds.
- `soroq-horizontal.svg` - compatibility wordmark for the current repo/site spelling.
- `shroq-mark-1024.png`, `shroq-mark-512.png`, `shroq-mark-256.png` - ready-to-upload raster exports.
- `shroq-horizontal-1440.png`, `shroq-horizontal-dark-1440.png`, `soroq-horizontal-1440.png` - raster wordmark exports for decks and social previews.
- `../../favicon.svg` - website favicon.

## Usage

Use the mark alone for favicons, app icons, avatars, and compact nav. Use a horizontal wordmark for website headers, pitch decks, docs covers, and social previews. Keep clear space around the mark equal to at least one quarter of its width.
